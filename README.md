# Sample Portfolio (Frontend)

This is a **frontend-only personal portfolio website** built to showcase skills in **HTML, Tailwind CSS, and JavaScript**.  

- Sections included: **Home, About, Projects, Contact**  
- Projects are **frontend demos only**  
- Contact form is **for demonstration**; no actual data is submitted  

## How to Use

1. Clone or download this repository.  
2. Open `index.html` in a web browser to view the portfolio.  

## Technologies

- HTML5  
- Tailwind CSS  
- JavaScript
